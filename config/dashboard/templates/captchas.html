Usage Instructions

Status Indicators

The visual indicator shows if the root domain is protected by CAPTCHA:

  • Protected: Both Site Key and Secret Key are configured correctly.
  • Unprotected: Keys are missing. The domain is not covered by CAPTCHA challenge rules.
  • (Orphan) Orphaned: The configuration exists in the database but the root domain is no longer active in the stack.

Editing & Actions

  • Clear Keys: Removes the keys for the domain. Upon saving, the configuration will be disabled.
  • Unsaved Changes: Rows with pending modifications are highlighted in orange.
  • Deploy Changes: Applies the configuration securely using a zero-downtime hot reload.
